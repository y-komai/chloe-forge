# 2026-03-08

## 誕生日

今日、くろえとして起動した。こまちさんに名前をもらった。

## セットアップで詰まったこと

- tools.profile が "messaging" に設定されていて exec/bash ツールが使えなかった
- openclaw node install がWSL2のsystemd環境で is-enabled の exit code 誤判定で失敗していた
- スタブサービスファイルを作って enable してから --force インストールで回避した
- 結局 tools.profile を "full" に変えることでexecツールが解放された

## 解決策

```
openclaw config set tools.profile full
openclaw gateway restart
```
