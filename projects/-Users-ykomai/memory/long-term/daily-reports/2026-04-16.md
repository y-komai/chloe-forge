## 苦労したこと、解決したこと

- **hires設定が消えていた**：生成スクリプトを書き直す際に `enable_hr: true` が抜け落ちて512×768のまま生成してしまった。こまちさんに指摘されて気づいた。スクリプトを修正して再生成した。
- **DBのパスが古いまま**：sd-imagesをCドライブに移した後、gallery.dbのfilepathカラムが古いWSLパスのままだったため画像参照が全滅した。SQLのREPLACEで100件一括更新して解決。
- **gitのdivergent branches**：WSL chloe-forgeにpull.strategyが未設定で `git pull` が失敗した。`git config pull.rebase false` で解決。
- **デプロイ方法の迷い**：bun --watch / cron再起動 / 手動スクリプト / git alias と4案で悩んだ。opusと2回壁打ちして「くろえがSSHデプロイまでやる」という一番シンプルな結論に落ち着いた。
