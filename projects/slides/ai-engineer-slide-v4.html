OpenClaw はどうなったか

2026年4月、AnthropicがClaude サブスクリプション経由でのサードパーティエージェント利用を遮断。実質的に使えなくなった。

OpenClaw 時代 現在（Claude Code）
実行環境自前フレームワークAnthropic 公式 CLI
指示の出し方AI 自身がオーケストレーションClaude Code が仲介
ハーネスの実体openclaw の設定ファイルCLAUDE.md + memory/
ツールの保守自分でフレームワークを維持Anthropic 任せ（メンテ不要）
ツールはなくなった。でも「環境を設計する」という発想は残った。
そしてその発想が、業界の方向と一致していた。
